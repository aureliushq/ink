* Static Site Generator [0/11]
  :PROPERTIES:
  :CATEGORY: ssg
  :END:
  Single-binary Go SSG: markdown content + HTML templates + CSS → static site.
  Config-driven, selectable built-in themes with per-site overrides, tags + series,
  built-in syntax highlighting, dev server with live reload.
